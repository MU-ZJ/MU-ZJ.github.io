A website which takes ingredients as an input, and presents the user with recipes using those ingredients. Ingredients needed for the recipe can be added to a shopping list, then removed as they are aqcuired. Once all ingredients are checked off, preparation instructions are provided.

Mars Rover Gallery

Website Link
Github Repository

A React project utilizing a NASA API to display images from the Mars rovers. The project pulls images directly from the NASA API, then loops through and inserts the image links into HTML cards utilizing Reacts’ useEffect and useState features.

Website Link
